🦆 **DaffyDocks** is a curated development environment that bundles common engineering tools and provides consistent defaults to reduce setup drift and "works on my machine" issues.

## 🔧 Supported Toolchains

- PHP
- JavaScript
- Python

## 🗄️ Backing Services

In addition to language toolchains, DaffyDocks supports common backing services used by modern applications, typically provisioned through Docker Compose profiles:

- Databases
- Caches
- Message brokers
- Search and indexing services

Service-specific configuration details will be documented in the installation guide.

## 🧰 Prerequisites

Before running this development environment, make sure Docker is installed and available:

- Docker Engine (or Docker Desktop)
- Docker Compose v2 (`docker compose`)

Quick verification:

```bash
docker --version
docker compose version
```
